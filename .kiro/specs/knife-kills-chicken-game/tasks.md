# 实现计划：刀杀鸡（扛刀扛枪版）卡牌游戏

## 概述

按照MVC-like架构，先实现核心游戏规则引擎（纯函数），再实现状态管理，然后实现AI决策，最后实现UI渲染和事件处理。关键变更：去掉强制吃子规则，新增扛刀/扛枪机制（人扛刀杀鸡、癞痢扛枪杀老虎），刀和枪不能主动移动，火箭克所有但同归于尽，PVE模式标识玩家方和电脑方，新增游戏规则面板。所有代码使用纯HTML/CSS/JavaScript，文件放置在 `apps/card-game/knife-kills-chicken/` 目录下。测试使用Vitest + fast-check。
